# Memory Pixel — Premium Interactive Showcases (`Showcase` Repository)

Welcome to the official showcase collection for **Memory Pixel** (`abhishek01032007-pixel`). 

This repository contains three complete, production-ready Next.js web application templates created by Abhishek M to demonstrate our premium design, animations, and emotional storytelling quality before clients place an order. All three templates are hosted natively using **GitHub Pages**.

## 🛠️ Running Locally
To run any of the three showcase templates on your computer:
```bash
# Example for Loved One Showcase
cd showcase-loved-one
npm install
npm run dev
```
Then open `http://localhost:3000` in your web browser.
